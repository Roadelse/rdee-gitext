# rdee-gitext

+ Roadelse series toolkit: Git extensions, mainly including two aspects:
    + Git subcommand expansion
    + Git hooks
    + Isolated executables
    + Shell auxiliary
+ Linux only

### Git sub-command
+ bin/git-url

### Git hooks
+ include/hooks/pre-commit
    - update VERSION automatically

### Isolated executables
+ bin/gits

### Shell auxiliary
+ lib/git-cd.sh
+ lib/git-prompt.sh
